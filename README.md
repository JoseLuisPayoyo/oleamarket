## 📖 **Sobre el Proyecto**

**OleaMarket** no es solo una tienda online de aceite de oliva. Es una **plataforma completa** que revoluciona la experiencia de compra, educación y conexión con la tradición olivarera mediante tecnología de vanguardia.

### 💎 **Diferenciadores Clave**

#### **🛒 Tienda E-commerce Premium**
- Catálogo completo de aceites de oliva virgen extra
- Múltiples variedades: Picual, Arbequina, Hojiblanca, Cornicabra
- Sistema de filtrado avanzado (intensidad, acidez, certificaciones)
- Comparador inteligente de productos
- Packs regalo personalizables
- Checkout optimizado con Stripe

#### **🌳 Adopta tu Olivo (EXCLUSIVO)**
- Adopta un olivo real, específico, con nombre y ubicación GPS
- Recibe el aceite de TU árbol cada temporada (12L/año)
- Dashboard personalizado con fotos y updates mensuales
- Certificado de adopción físico + placa en el árbol
- Invitación anual a visitar y cosechar tu olivo

#### **🔄 Club de Suscripción "El Olivar"**
- **Nivel Explorador** (19,90€/mes): 1 botella mensual diferente
- **Nivel Catador** (39,90€/mes): 2 botellas + catas online
- **Nivel Maestro** (79,90€/mes): 3 botellas + visita almazara
- Modo "Sorpréndeme" o "Yo Elijo"

#### **🎁 Constructor de Packs Regalo**
- Interfaz drag & drop para crear packs personalizados
- Elección de caja (madera, cartón premium, cestería)
- Vista previa 3D del pack
- Tarjeta personalizada incluida
- Sistema de descuentos por volumen

#### **🎓 Academia del Aceite (SEO Gold)**
- Cursos interactivos sobre variedades y catas
- Videos educativos tipo TikTok/Reels
- Quiz con certificado descargable
- Blog con recetas y maridajes
- Glosario técnico interactivo

## 🌟 **Características Innovadoras Completas**

### **1. 🗺️ Trazabilidad Total**
Cada botella tiene un código QR que muestra:
- Finca de origen con mapa interactivo
- Fecha exacta de recolección
- Temperatura de molturación
- Análisis químico completo (acidez, peróxidos, K270)
- Timeline del proceso productivo
- Condiciones climáticas de la cosecha

### **2. 🎯 Perfil de Paladar Personalizado**
Quiz inteligente que:
- Analiza preferencias de sabor (suave, intenso, picante)
- Detecta hábitos culinarios
- Recomienda aceites según perfil
- Guarda historial de compras
- Sugiere recetas personalizadas

### **3. 📊 Comparador Inteligente de Aceites**
Herramienta visual para comparar hasta 3 aceites:
- Gráfica de araña con características organolépticas
- Comparación de análisis químico
- Precio normalizado por ml
- Recomendación de uso para cada uno
- Certificaciones y premios

### **4. 🎁 Constructor de Packs Personalizado**
Sistema innovador donde el cliente:
- Arrastra y suelta productos en la caja
- Elige tipo de empaque (madera, cartón, cesta)
- Ve vista previa 3D en tiempo real
- Añade tarjeta con mensaje personalizado
- Obtiene descuentos automáticos por volumen

### **5. 🔄 Sistema de Suscripciones**
Tres niveles con beneficios crecientes:
- Aceites diferentes cada mes
- Opción curación automática o elección manual
- Acceso a catas online exclusivas
- Descuentos progresivos en tienda
- Productos de edición limitada

### **6. 🎓 Academia del Aceite**
Contenido educativo que mejora SEO:
- Módulos: "Cómo catar aceite profesionalmente"
- "Diferencias entre variedades"
- "Mitos y verdades del AOVE"
- Certificados descargables
- Miles de palabras clave long-tail

### **7. 🏆 Sistema de Gamificación**
Niveles de "Experto Oleícola":
- 🌱 Novato → 🌿 Aficionado → 🫒 Conocedor → 🏺 Maestro → 👑 Sommelier
- Puntos canjeables por productos
- Badges y logros
- Acceso anticipado a ediciones limitadas
- Descuentos exclusivos por nivel

### **8. 💬 Asistente Virtual "Oliva"**
Chatbot especializado que:
- Responde dudas sobre aceites
- Recomienda según el plato
- Explica certificaciones (DOP, Ecológico)
- Ayuda en el proceso de compra
- Disponible 24/7

### **9. 🌐 Experiencias y Turismo Oleícola**
Reserva online de:
- Visitas guiadas a la almazara
- Catas presenciales con expertos
- Jornadas de recolección participativa
- Talleres de cocina con aceite
- Calendario integrado de disponibilidad

### **10. 📸 Galería Social #MiAceite**
Comunidad activa donde:
- Clientes suben fotos de platos con tu aceite
- Sistema de votación mensual
- Premio al mejor plato del mes

## 🚀 **Inicio Rápido**

### **Prerequisitos**

- Node.js 20+
- Java 17+
- Docker & Docker Compose
- Git
- Maven (incluido en proyecto)

### **Instalación Local**
```bash
# 1. Clonar repositorio
git clone https://github.com/tu-usuario/oleamarket.git
cd oleamarket

# 2. Iniciar servicios con Docker
docker-compose up -d

# Verificar que están corriendo:
docker ps
# Deberías ver: oleamarket-db, oleamarket-redis, oleamarket-pgadmin

# 3. Backend - Spring Boot
cd backend
./mvnw spring-boot:run

# 4. Frontend - Next.js (en otra terminal)
cd frontend
npm install
npm run dev
```

### **URLs Locales**

| Servicio | URL | Credenciales |
|----------|-----|--------------|
| 🎨 Frontend | http://localhost:3000 | - |
| 🔧 Backend API | http://localhost:8080 | - |
| 📊 Swagger UI | http://localhost:8080/swagger-ui.html | - |
| 🐘 PostgreSQL | localhost:5432 | postgres/postgres |
| 🗄️ pgAdmin | http://localhost:5050 | admin@oleamarket.com/admin |
| 🔴 Redis | localhost:6379 | - |

### **Variables de Entorno**
```bash
# Backend - application-dev.yml
SPRING_PROFILES_ACTIVE=dev

# Frontend - .env.local
NEXT_PUBLIC_API_URL=http://localhost:8080/api
```

## 🗄️ **Modelo de Datos Principal**
```
User (Cliente/Admin)
  ├──< Adopcion >──┤ Olivo
  ├──< Order >──┤ Product
  ├──< Subscription
  ├──< Cart >──┤ CartItem
  ├──< Review
  ├──< Address
  └──< UserProfile (Perfil de paladar)

Olivo
  ├── Variedad (Picual, Arbequina, etc.)
  ├── Finca (Location, historia)
  ├──< TimelineEvent (Updates, fotos)
  ├──< Adopcion
  └── disponible (boolean)

Product (Aceite en tienda normal)
  ├── Categoria
  ├── Variedad
  ├── Intensidad, Acidez, etc.
  ├──< OrderItem
  ├──< Review
  └──< ProductImage

Order
  ├──< OrderItem
  ├── Payment (Stripe)
  ├── ShippingAddress
  └── Status (enum)

Subscription
  ├── Plan (Explorador, Catador, Maestro)
  ├── Status (active, cancelled, paused)
  ├── User
  └── NextBillingDate

CustomPack (Packs personalizados)
  ├── User
  ├──< PackItem >──┤ Product
  ├── BoxType (madera, cartón, cesta)
  ├── PersonalMessage
  └── TotalPrice

Article (Blog/Academia)
  ├── Category
  ├── Tags
  ├── Author
  └── PublishedAt

Experience (Turismo oleícola)
  ├── Type (visita, cata, cosecha, taller)
  ├── AvailableDates
  ├── MaxParticipants
  └── Price
```

```bash
git push origin develop
# → GitHub Actions ejecuta tests
# → Railway despliega backend automáticamente
# → Vercel despliega frontend automáticamente

# Deploy a Producción
git checkout main
git merge develop
git push origin main
# → GitHub Actions ejecuta tests
# → Railway despliega a producción

```

## 📝 **Convenciones de Código**

### **Commits (Conventional Commits)**
```
feat(scope): descripción corta
fix(scope): descripción corta
docs(scope): descripción corta
style(scope): descripción corta
refactor(scope): descripción corta
test(scope): descripción corta
chore(scope): descripción corta
```

**Ejemplos:**
```bash
git commit -m "feat(auth): add JWT authentication with refresh tokens"
git commit -m "feat(olivo): implement adoption system"
git commit -m "fix(cart): prevent duplicate items in shopping cart"
git commit -m "docs(readme): update installation instructions"
git commit -m "refactor(user): extract validation logic to service layer"
```

### **Branches**
```
main              → Producción
develop           → Integración
feature/*         → Nuevas funcionalidades
bugfix/*          → Corrección de bugs
hotfix/*          → Fixes urgentes en producción
release/*         → Preparación de releases
```
